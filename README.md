# React Native - The Practical Guide [2025]

Bem-vindo ao repositório do curso **React Native - The Practical Guide [2025]**! 🚀

Este repositório contém diversos projetos práticos desenvolvidos durante o curso de React Native, cobrindo desde conceitos fundamentais até funcionalidades avançadas da plataforma.

## 📋 Conteúdo do Repositório

O repositório está organizado em vários diretórios, cada um contendo um projeto completo com foco em diferentes aspectos do desenvolvimento com React Native:

### Projetos Inclusos

- **device-features** - Exploração de recursos do dispositivo
- **expense-tracker** - Aplicação de rastreamento de despesas
- **game** - Aplicação de jogo interativo
- **goals-app** - Aplicativo de gerenciamento de metas/objetivos
- **meals-app** - Aplicação de receitas e refeições

## 🛠️ Tecnologias Utilizadas

- **React Native** - Framework para desenvolvimento mobile multiplataforma
- **JavaScript** - Linguagem de programação utilizada em todo o projeto
- **Expo** - Plataforma para desenvolvimento e construção de apps React Native
- **Outros** - Bibliotecas e ferramentas específicas de cada projeto

## 🚀 Como Começar

### Pré-requisitos

- Node.js instalado na sua máquina
- npm ou yarn como gerenciador de pacotes
- Expo CLI (opcional, mas recomendado)

```bash
npm install -g expo-cli
```

### Instalação e Execução

1. Clone o repositório:
```bash
git clone https://github.com/jocsamendonca/React-Native---The-Practical-Guide-2025-
cd React-Native---The-Practical-Guide-2025-
```

2. Acesse o diretório do projeto desejado:
```bash
cd [nome-do-projeto]
```

3. Instale as dependências:
```bash
npm install
# ou
yarn install
```

4. Execute o projeto:
```bash
npx expo start
# ou
npm start
```

5. Escaneie o código QR com o Expo Go no seu dispositivo mobile ou use um emulador

Cada projeto segue uma estrutura padrão de aplicação React Native:

```

```

## 📖 Descrição dos Projetos

### Device Features
Aplicação que demonstra o uso de recursos nativos do dispositivo como câmera, galeria, sensores e permissões.

### Expense Tracker
Um gerenciador de despesas que permite criar, editar e deletar transações, com visualização de gráficos e resumos.

### Game
Aplicação de jogo interativo desenvolvida em React Native, demonstrando conceitos de animação e interação do usuário.

### Goals App
Aplicativo para criar e gerenciar metas/objetivos pessoais com progresso e notificações.

### Meals App
Aplicação de receitas que permite buscar, filtrar e salvar receitas favoritas, com detalhes nutricionais.
